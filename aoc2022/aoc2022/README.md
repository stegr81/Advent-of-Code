# 2022-Advent-of-Code
#### [Day 1](https://adventofcode.com/2022/day/1):
###### Part 1 - Each line contains a number representing calories for a food item. Blocks separated by a blank line indicate new group (elf). Calculate how many calories the top elf is carrying.
###### Part 2 - Using the same list, calculate the number of calories carried by the top 3 elves.

#### [Day 2](https://adventofcode.com/2022/day/2):
###### Part 1 - Calculate the number of points achieved based on a variety of parameters whilst playing Rock,Paper,Scissors using a'strategy guide'.
###### Part 2 - Revised calculation based on new information.

#### [Day 3](https://adventofcode.com/2022/day/3):
###### Part 1 - A string of characters represents 2 compartments. Calculate where duplicates exist in each compartment over a series of strings.
###### Part 2 - For each group of 3 strings, identify the unique value common to each.

#### [Day 4](https://adventofcode.com/2022/day/4)
###### Part 1 - Each line of text contains a pair of comma separated number ranges; determine which lines have ranges completely contained within the other.
###### Part 2 - With the same input; determine if there is any overlap in the number ranges.

#### [Day 5](https://adventofcode.com/2022/day/5)
###### Part 1 - A text input provides configuration for stacks of crates. Move the crates, one at a time, between stacks based on instructions and provide the top crate in each stack.
###### Part 2 - Using the same input, the crates can now be moved in one go, thus retaining their order. Provide the top crate in each stack.

#### [Day 6](https://adventofcode.com/2022/day/6)
###### Part 1 - Given a string of characters, how many characters must pass before you have a sequence of 4 unrepeated characters.
###### Part 2 - Using the same input, how many characters must pass before you have a sequence of 14 unrepeated characters.

#### [Day 7](https://adventofcode.com/2022/day/7)
###### Part 1 - Given a text file of linux commands for navigating a folder structure, calculate the sum of all directories larger than 100000
###### Part 2 - Knowing the total storage space, the space used and the total available to run a system update, find the size of the smallest folder you could delete to free the required space

#### [Day 8](https://adventofcode.com/2022/day/8)
###### Part 1 - Receiving a grid of numbers representing the height of an area of woodland, calculate the trees visible from the outside of the woodland.
###### Part 2 - Calculate the scenic score for each given tree - that is the product of the number of trees visible from each individual tree
